程序执行方式：
1.进入src目录
2.编译：javac Main.java
3.运行：java Main
4.程序会读取random.txt中的数据，分别用六种方式进行排序，将排序后的数据写入order1.txt, order2.txt, order3.txt, order4.txt, order5.txt, order6.txt，
排序结束后会在终端打印六种算法的运行时间。

并行快排执行方式：
并行快排类接受数据之后，每次partition之后都会检查数据大小，如果小于阈值直接使用串行快排方式，
如果大于等于阈值，在规定的线程数范围内，会把数据分成两份交给两个线程处理。

并行枚举排序执行方式：
并行枚举排序会直接把数据分成8组，然后对每一组做串行的枚举排序

并行归并排序执行方式：
并行归并排序接受到数据之后检查数据大小，如果小于阈值直接使用串行的归并排序方式，
否则在规定的线程数范围内，把数据分成两部分交给两个线程处理，处理之后再merge。
